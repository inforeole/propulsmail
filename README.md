[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Finforeole%2Fpropulsmail&project-name=propulsmail&repository-name=propulsmail&products=%5B%7B%22type%22%3A%22integration%22%2C%22integrationSlug%22%3A%22convex%22%2C%22productSlug%22%3A%22convex%22%2C%22protocol%22%3A%22storage%22%7D%5D)

### Ce que fait ce bouton

1. Crée un compte Vercel (gratuit) si vous n'en avez pas
2. Copie le projet dans votre GitHub
3. Connecte automatiquement la base de données Convex
4. Met en ligne votre site

### Mises à jour automatiques

Après le deploy, activez les mises à jour automatiques :

1. Allez sur votre repo GitHub (celui créé par Vercel)
2. Cliquez sur l'onglet **Actions**
3. Cliquez sur **I understand my workflows, go ahead and enable them**

C'est tout. Votre app se mettra à jour automatiquement toutes les 6 heures.

### Pré-requis

- Un compte GitHub (gratuit) — [en créer un](https://github.com/signup)
- C'est tout.
